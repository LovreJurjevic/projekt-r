# Projekt R

## Struktura repozitorija projekta

+ ```fastener_dataset/```: Neizmijenjeni zadani direktorij sa skupom podataka (zbog svoje memorijske veličine nije u repozitoriju)
+ ```models/```: Istrenirani modeli, označeni brojevima (U repozitoriju su jedino modeli s veličinom manjom od 100 MB)
+ ```notebooks/```: Sve Jupyter bilježnice korištene u projektu
+ ```pdfs/```: PDF datoteke spremljene kako bi se sačuvali podaci o treniranju (kasnije sam se odlučio na spremanje jedino plota)
+ ```src/```: Modul napravljen kako bi se u bilježnici na lakši način mogao koristiti kod iz prošlih bilježnica
