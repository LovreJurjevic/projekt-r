# Modeli istrenirani tokom projekta

## Opis modela

+ ```01/```: Nastao eksperimentiranjem nad prvim ResNet modelom, uglavnom radi optimizacije učenja (ne sadrži nikakvu augmentaciju podataka)
+ ```02/```: Nastao treniranjem Miličićevog 'jednostavnog' modela, uz preinake u načinu učenja
+ ```03/```: Konvolucijska mreža slične kompleksnosti kao Miličićev 'jednostavni' model, uz nekoliko razlika (~96% točnost)
+ ```04/```: Ispočetka napravljena ResNet mreža i uspješno istrenirana (~95% točnost)
